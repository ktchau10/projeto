Boas-vindas e Acesso

Seja bem-vindo ao escritório digital da Acessibilidade na UFOPA. O Módulo NEE foi criado para que você não precise mais usar planilhas soltas. Aqui, centralizamos toda a vida do discente que precisa de apoio.

Para que serve?

  • Registrar: Saber exatamente quem são os alunos e quais suas necessidades (Ex: Baixa visão, TEA, etc).
  • Acompanhar: Criar um histórico de todos os atendimentos, visitas e entrevistas.
  • Gerenciar: Controlar quem recebe bolsa e quem está na fila de espera.

Como entrar no sistema:

  1. Faça seu login normal no SIGAA.
  2. No menu principal, procure a opção Módulos (ícone de blocos/quadrados).
  3. Clique em NEE - Necessidades Educacionais Especiais.
Figura 1: Acesso via Menu Módulos

Configurações Iniciais (Administração)

Antes de cadastrar qualquer aluno, o sistema precisa saber quais tipos de deficiência ou condições o núcleo atende. Esta é uma tarefa para o Gestor.

Quando usar: Use esta tela sempre que precisar incluir uma nova terminologia (Ex: "Altas Habilidades", "Mobilidade Reduzida") na lista de opções.

Passo a passo:

  1. Vá em: Administração → Tipos de Necessidades Educacionais Específicas → Cadastrar.
  2. No campo Denominação, digite o nome da necessidade.
  3. Clique no botão verde Cadastrar. Pronto, agora ela aparecerá na lista para vincular aos alunos.
Figura 2: Tela de Cadastro

Errou a digitação ou uma categoria não é mais usada? Você pode ajustar aqui.

Passo a passo:

  1. Vá em: Administração → Tipos de Necessidades Educacionais Específicas → Listar/Alterar.
  2. Use a busca para encontrar o item.
  3. Escolha uma ação:
    • Clique no Lápis para corrigir o nome.
    • Clique na Lixeira para apagar.
Dica: O sistema protege os dados. Você só conseguirá apagar uma categoria se nenhum aluno estiver vinculado a ela.
Figura 3: Lista de Necessidades

Gerenciando Solicitações e Acompanhamento

É aqui que o trabalho começa. Quando um coordenador ou aluno pede apoio pelo portal, o pedido chega nesta lista para você analisar.

O que fazer:

  1. Vá em: Alunos → Discentes com NEE → Novas Solicitações de Apoio.
  2. Você verá uma lista com os pedidos pendentes.
  3. Clique no ícone de Lupa ou Prancheta ao lado do nome do aluno para abrir os detalhes.
Figura 4: Caixa de Entrada de Solicitações

Como dar o Parecer (Decisão): Ao abrir o pedido, leia a justificativa e decida:

  • Deferido: Escolha esta opção para ACEITAR o pedido. O aluno entra para o cadastro ativo.
  • Indeferido: Escolha esta opção para NEGAR o pedido (necessário justificar).
  • Clique em Confirmar para finalizar. O aluno receberá um e-mail automático com a resposta.
Figura 5: Tela de Parecer Técnico

Nem sempre o aluno faz o pedido pelo portal. Se ele vier presencialmente ao setor, você deve cadastrá-lo manualmente por aqui. Também use esta tela para desativar alunos que formaram ou trancaram.

Passo a passo:

  1. Vá em: Alunos → Discentes com NEE → Gerenciar Estudante com NEE.
  2. Digite o nome ou matrícula na busca para ver se ele já existe.
  3. Se não existir, clique no botão Cadastrar Novo.
  4. Se precisar remover o vínculo do aluno com o setor, use o ícone da Lixeira.
Figura 6: Gerenciar Estudante

Esta é a parte mais importante para a memória do setor. Tudo o que for feito com o aluno deve ser registrado aqui.

Como registrar:

  1. Vá em: Alunos → Discentes com NEE → Avaliação e Acompanhamento.
  2. Busque o aluno.
  3. Preencha o formulário:
    • Tipo: Foi uma visita técnica? Uma entrevista? Entrega de material?
    • Descrição: Escreva um resumo do que aconteceu.
  4. Salve. Isso cria um histórico cronológico que qualquer técnico autorizado poderá consultar no futuro.
Figura 7: Registro de Acompanhamento

Precisa saber se o aluno está indo às aulas ou se as notas melhoraram, mas não quer sair do módulo NEE? Use esta ferramenta de consulta rápida.

O que você vai ver:

  • As turmas matriculadas no semestre atual.
  • Os horários das aulas.
  • As notas lançadas pelos professores até o momento.

Dados Detalhados

O cadastro do aluno só fica completo quando dizemos ao sistema o que ele tem. Um aluno pode ter mais de uma necessidade (Ex: Ser "Cego" e ter "Diabetes").

Como fazer:

  1. Vá em: Alunos → Dados do Discente → Gerenciar NEE de Discente.
  2. Selecione o aluno.
  3. Na lista "Necessidade", escolha a condição dele e clique em Adicionar.
  4. Repita se houver mais de uma.
Figura 9: Vinculando Necessidades

Precisa anotar algo sensível que não deve aparecer para o aluno, apenas para a equipe da PROGES?

  1. Vá em: Alunos → Dados do Discente → Editar Observações.
  2. Escreva no campo de texto.
Atenção: Tudo o que for escrito aqui é restrito à equipe técnica do NEE. Use com ética e responsabilidade.
Figura 10: Campo de Observações

Comunicação

Precisa dar um aviso geral sobre renovação de bolsas ou um evento? Não use seu e-mail pessoal. Use o sistema para garantir oficialidade.

  1. Vá em: Alunos → Notificar Discentes.
  2. Escolha quem recebe: Um aluno só, todos de um curso ou todos os alunos NEE.
  3. Escreva o assunto e a mensagem. O sistema envia para o e-mail cadastrado e para o Portal do Aluno.
Figura 11: Envio de Notificações

Relatórios e Estatísticas

Esta ferramenta serve para tomadas de decisão. Você pode gerar listas em PDF ou planilhas Excel para trabalhar os dados.

Exemplos de uso:

  • "Quantos alunos com TEA temos hoje?" -> Filtre por Necessidade: Transtorno do Espectro Autista.
  • "Quem está aguardando análise?" -> Filtre por Situação: Em Avaliação.
  • "Quero uma lista de todos os bolsistas." -> Filtre por Situação: Em Atendimento (ou Bolsista, se houver).
