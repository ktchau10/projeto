Introdução e Acesso

O Módulo de Necessidades Educacionais Especiais (NEE) é a ferramenta institucional da UFOPA desenvolvida para centralizar o acompanhamento pedagógico e administrativo dos discentes que possuem algum tipo de deficiência, transtorno global do desenvolvimento ou altas habilidades.

  • Finalidade: Permitir o registro detalhado das condições específicas de cada estudante e gerenciar bolsas.
  • Público: Gestores da PROGES, Coordenadores, Docentes e Alunos.

Para iniciar suas atividades, realize o login no sistema integrado e siga o caminho abaixo:

SIGAA → Módulos → NEE - Necessidades Educacionais Especiais
[INSIRA O PRINT AQUI: Tela inicial do SIGAA com destaque no menu Módulos]
Figura 1: Acesso aos Módulos

Administração (Configurações)

Esta funcionalidade permite registrar novas categorias de deficiências, transtornos ou altas habilidades no sistema.

Módulos → NEE → Administração → Tipos de Necessidades → Cadastrar
[INSIRA O PRINT AQUI: Formulário de cadastro de necessidade]
Detalhamento da tela:
  • Denominação: Digite o nome completo da necessidade (Ex: Transtorno do Espectro Autista).
  • Botão Cadastrar: Salva a nova necessidade no banco de dados.

Utilize esta opção para corrigir nomes de necessidades ou consultar as categorias ativas.

Módulos → NEE → Administração → Tipos de Necessidades → Listar/Alterar
[INSIRA O PRINT AQUI: Lista de necessidades cadastradas]
  • Ícone Lápis: Editar o nome.
  • Ícone Lixeira: Excluir (apenas se não houver alunos vinculados).

Gestão de Discentes

Aqui ficam listados todos os pedidos de apoio abertos por alunos ou coordenadores aguardando parecer.

Módulos → NEE → Alunos → Discentes com NEE → Novas Solicitações
[INSIRA O PRINT AQUI: Listagem de solicitações pendentes]

Clique no ícone de Lupa ou Prancheta para analisar o pedido.

Tela de Parecer:
[INSIRA O PRINT AQUI: Tela de preenchimento do parecer]
  • Parecer: Escreva a decisão técnica detalhada.
  • Situação: Selecione DEFERIDO (Aceito) ou INDEFERIDO (Negado).

Utilize esta funcionalidade para cadastrar manualmente um aluno que não fez solicitação via sistema ou para inativar vínculos.

Módulos → NEE → Alunos → Discentes com NEE → Gerenciar Estudante
[INSIRA O PRINT AQUI: Tela de busca e cadastro de estudante]
  • Botão Cadastrar: Inicia um novo vínculo.
  • Ícone Lixeira: Remove o vínculo do aluno com o NEE.

Fundamental para o histórico. Registre aqui todas as visitas, entrevistas e entregas de materiais.

Módulos → NEE → Alunos → Discentes com NEE → Avaliação e Acompanhamento
[INSIRA O PRINT AQUI: Formulário de registro de acompanhamento]
  • Tipo de Acompanhamento: Ex: Visita Técnica, Entrevista Psicológica.
  • Descrição: Relate o que foi observado.

Visualização rápida do desempenho do aluno (notas, turmas, horários) sem sair do módulo.

Módulos → NEE → Alunos → Discentes com NEE → Acompanhamento Acadêmico
[INSIRA O PRINT AQUI: Tela de resumo acadêmico]

Dados do Discente

Acesso à ficha completa do aluno e campo para notas internas da equipe.

Módulos → NEE → Alunos → Dados do Discente → Consultar / Editar Observações
[INSIRA O PRINT AQUI: Ficha do aluno e campo de observação]

As observações são de uso interno e não aparecem para o aluno.

Tela para associar as categorias (deficiências) cadastradas ao perfil do aluno.

Módulos → NEE → Alunos → Dados do Discente → Gerenciar NEE
[INSIRA O PRINT AQUI: Tela de vínculo de necessidade]
  • Necessidade: Selecione na lista e clique em "Adicionar".
  • Um aluno pode ter múltiplas necessidades vinculadas.

Comunicação

Envio de mensagens em massa ou individuais para o Portal do Aluno e E-mail.

Módulos → NEE → Alunos → Notificar Discentes
[INSIRA O PRINT AQUI: Tela de composição de mensagem]

Relatórios Gerenciais

Permite extrair listagens completas aplicando filtros específicos para análise estatística.

Módulos → NEE → Relatórios → Discentes → Relatório Geral de Discentes com NEE
[INSIRA O PRINT AQUI: Tela de filtros do relatório]
Filtros Principais:
  • Tipos de Necessidades: Filtre por "Todas" ou categorias específicas (ex: Baixa Visão).
  • Situação do Atendimento:
    • Em Avaliação: Aguardando parecer.
    • Em Atendimento: Alunos ativos.
  • Formato: Escolha entre PDF (impressão) ou XLS (Excel para edição).
