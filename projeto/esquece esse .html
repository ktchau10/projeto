Introdução ao Módulo

O Módulo de Necessidades Educacionais Especiais (NEE) é a ferramenta institucional da UFOPA desenvolvida para centralizar o acompanhamento pedagógico e administrativo dos discentes que possuem algum tipo de deficiência, transtorno global do desenvolvimento ou altas habilidades.

Sua principal finalidade, diferentemente do cadastro comum de alunos, é permitir o registro detalhado das condições específicas de cada estudante, como o tipo de apoio necessário para sua permanência.

Para iniciar suas atividades e ter acesso às funcionalidades de gestão, você deve realizar o login no sistema integrado e navegar até o módulo específico.

Caminho: SIGAA → Módulos → NEE - Necessidades Educacionais Especiais

[Imagem: Tela inicial do SIGAA com destaque no menu Módulos]

Administração e Configurações

Esta funcionalidade permite registrar no sistema novas categorias de deficiências, transtornos ou altas habilidades.

Caminho: Módulos → NEE → Administração → Tipos de Necessidades Educacionais Específicas → Cadastrar

[Imagem: Formulário de cadastro de necessidade]
  • Denominação: Digite o nome completo da necessidade (Ex: Baixa Visão).
  • Botão Cadastrar: Salva a nova necessidade.

Gestão de Discentes

Esta é a "Caixa de Entrada" do Gestor. Aqui ficam listados todos os pedidos de apoio abertos.

Caminho: Módulos → NEE → Alunos → Discentes com NEE → Novas Solicitações de Apoio

[Imagem: Lista de solicitações pendentes]

Ação: Clique no ícone de lupa para analisar o pedido e emitir o parecer (Deferido/Indeferido).

Utilize esta tela para registrar o histórico de atendimentos (entrevistas, visitas).

Caminho: Módulos → NEE → Alunos → Discentes com NEE → Avaliação e Acompanhamento

[Imagem: Tela de registro de acompanhamento]

É nesta tela que vinculamos a deficiência específica ao aluno.

Caminho: Módulos → NEE → Alunos → Dados do Discente → Gerenciar NEE de Discente

Relatórios Gerenciais

Permite extrair listagens completas dos estudantes cadastrados.

Caminho: Módulos → NEE → Relatórios → Discentes → Relatório Geral
